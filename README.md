## üöÄ How to Use

### For Students/Users:

1. **Browse the Repository**
   - Navigate through the folders to find the specific lab exercise or algorithm you're interested in
   - Each notebook (`.ipynb` file) contains a complete lab exercise with explanations and code

2. **Open in Google Colab**
   - Click on any `.ipynb` notebook file in this repository
   - Click the "Open in Colab" badge at the top of the notebook, OR
   - Click the "Open in Colab" button that appears in the GitHub preview
   - Alternatively, you can manually open Google Colab and import the notebook using the GitHub tab

3. **Run the Notebook**
   - Once opened in Colab, you can run cells individually by clicking the play button (‚ñ∂Ô∏è) next to each cell
   - Or run all cells at once: `Runtime` ‚Üí `Run all`
   - Make sure you're signed in to your Google account to save any changes

4. **Make Your Own Copy**
   - To experiment and save changes: `File` ‚Üí `Save a copy in Drive`
   - This creates a personal copy in your Google Drive that you can modify freely

### For Contributors:

1. **Fork the Repository**
   ```bash
   # Clone your fork
   git clone https://github.com/YOUR-USERNAME/Algorithms-and-Problem-Solving-Lab.git
   cd Algorithms-and-Problem-Solving-Lab
   ```

2. **Create/Edit Notebooks**
   - Open Google Colab and create/edit your notebook
   - Download the notebook: `File` ‚Üí `Download` ‚Üí `Download .ipynb`

3. **Add to Repository**
   ```bash
   # Add your notebook
   git add your-notebook.ipynb
   git commit -m "Add: [description of lab exercise]"
   git push origin main
   ```

4. **Create Pull Request**
   - Open a pull request to contribute your work back to the main repository
